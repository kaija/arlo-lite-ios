概述

Arlo Lite（以下稱「我們」、「本應用程式」）是一款免費、開源的 iOS 應用程式，用於與大型語言模型（LLM）進行互動。我們致力於保護您的隱私。本政策說明本應用程式處理哪些資料、如何儲存，以及哪些資料絕不會被蒐集。

我們不蒐集的資料

  • 我們不會蒐集、傳輸或在外部伺服器上儲存任何個人資訊。
  • 我們不使用任何形式的分析工具、遙測或追蹤技術。
  • 我們沒有後端伺服器——不存在伺服器端元件。
  • 我們不會存取、讀取或與任何第三方分享您的對話內容。
  • 我們不顯示任何廣告。

儲存在您裝置上的資料

Arlo Lite 產生的所有資料均保留在您的裝置上：

  • 對話記錄——使用裝置上的 SQLite 資料庫進行本地儲存。
  • API 金鑰——僅儲存在 iOS Keychain（Apple 的加密憑證存放區）中，絕不會以明文寫入檔案或未加密的儲存空間。
  • 服務商設定——本地儲存於裝置上（模型選擇、基礎 URL、顯示名稱）。
  • 使用者偏好設定——主題、語言、系統提示詞及其他設定均儲存於本地。

網路通訊

Arlo Lite 僅在以下情況下進行網路通訊：

  • LLM API 呼叫——當您發送訊息時，應用程式會直接向您設定的 LLM 服務商（如 OpenAI、Anthropic 或您的自訂端點）發出 HTTPS 請求。您的對話內容直接傳送給您選擇的服務商，我們不參與此通訊過程。
  • 模型中繼資料——應用程式可能會從公開 CDN 取得模型定價及上下文視窗大小等資訊，以保持模型資訊的即時性。

您有責任查閱所選 LLM 服務商的隱私權政策，了解他們如何處理您的資料。

iCloud 備份

若您在應用程式中啟用 iCloud 備份：

  • 對話文字記錄與服務商設定（不含 API 金鑰）可能會備份至您的 iCloud 帳號。
  • API 金鑰絕不會包含在 iCloud 備份中。
  • 檔案或圖片附件不會同步。
  • iCloud 資料受 Apple 隱私權政策及您的 iCloud 帳號設定管轄。

第三方服務

本應用程式未整合任何用於分析、當機報告或廣告的第三方 SDK。應用程式僅與以下外部服務通訊：

  • 您明確設定的 LLM 服務商（OpenAI、Anthropic 或自訂端點）。
  • Apple 的 iCloud 服務（僅在您啟用備份時）。

兒童隱私

Arlo Lite 並非面向 13 歲以下兒童。我們不會刻意蒐集任何兒童的資料。

政策變更

我們可能會不定期更新本隱私權政策。變更將反映在本頁面上，並更新「最後更新日期」。由於本應用程式不蒐集任何資料，預期變更幅度極小。

聯絡方式

若您對本隱私權政策有任何疑問，請在我們的 GitHub 儲存庫上提出 Issue。

開放原始碼

Arlo Lite 是開源軟體。您可以在 github.com/kaija/arlo-lite 查閱完整原始碼，以驗證上述隱私聲明。
