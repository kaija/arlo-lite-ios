概要

Arlo Lite（以下「当アプリ」）は、大規模言語モデル（LLM）と対話するための無料・オープンソースの iOS アプリケーションです。私たちはお客様のプライバシー保護に取り組んでいます。本ポリシーでは、当アプリが取り扱うデータ、その保存方法、および収集しないデータについて説明します。

収集しないデータ

  • 個人情報を外部サーバーに収集、送信、保存することはありません。
  • アナリティクス、テレメトリー、トラッキングは一切使用しません。
  • バックエンドサーバーは存在しません — サーバーサイドのコンポーネントはありません。
  • お客様の会話にアクセス、閲覧、または第三者と共有することはありません。
  • 広告は表示しません。

デバイスに保存されるデータ

Arlo Lite で生成されるすべてのデータはお客様のデバイス上に留まります：

  • チャット履歴 — デバイス上の SQLite データベースにローカル保存されます。
  • API キー — iOS Keychain（Apple の暗号化された資格情報ストレージ）にのみ保存されます。平文ファイルや暗号化されていないストレージには一切書き込まれません。
  • プロバイダー設定 — デバイス上にローカル保存されます（モデル選択、ベース URL、表示名）。
  • ユーザー設定 — テーマ、言語、システムプロンプトなどの設定はローカルに保存されます。

ネットワーク通信

Arlo Lite がネットワーク通信を行うのは以下の場合のみです：

  • LLM API 呼び出し — メッセージを送信すると、設定された LLM プロバイダー（OpenAI、Anthropic、またはカスタムエンドポイント）に直接 HTTPS リクエストを送信します。会話内容はお客様が選択したプロバイダーに直接送られ、当アプリはこの通信に関与しません。
  • モデルメタデータ — モデル情報を最新に保つため、公開 CDN からモデルの価格やコンテキストウィンドウサイズなどの情報を取得することがあります。

選択した LLM プロバイダーがお客様のデータをどのように扱うかについては、各プロバイダーのプライバシーポリシーをご確認ください。

iCloud バックアップ

アプリ内で iCloud バックアップを有効にした場合：

  • チャットテキストとプロバイダー設定（API キーを除く）が iCloud アカウントにバックアップされる場合があります。
  • API キーは iCloud バックアップに含まれません。
  • ファイルや画像の添付ファイルは同期されません。
  • iCloud データは Apple のプライバシーポリシーおよびお客様の iCloud アカウント設定に準拠します。

サードパーティサービス

当アプリは、アナリティクス、クラッシュレポート、広告のためのサードパーティ SDK を一切組み込んでいません。アプリが通信する外部サービスは以下のみです：

  • お客様が明示的に設定した LLM プロバイダー（OpenAI、Anthropic、またはカスタムエンドポイント）。
  • Apple の iCloud サービス（バックアップを有効にした場合のみ）。

お子様のプライバシー

Arlo Lite は 13 歳未満のお子様を対象としていません。お子様のデータを故意に収集することはありません。

ポリシーの変更

本プライバシーポリシーは随時更新される場合があります。変更はこのページに反映され、「最終更新日」が更新されます。当アプリはデータを収集しないため、変更は最小限にとどまる見込みです。

お問い合わせ

本プライバシーポリシーについてご質問がある場合は、GitHub リポジトリで Issue を作成してください。
