Arlo Lite는 LLM과 대화하기 위한 경량 iOS 클라이언트입니다. 자신의 API 키를 사용하여 프로바이더에 직접 연결 — 백엔드 없음, 구독 없음, 텔레메트리 없음.

GitHub에서 보기 자세히 보기

파워 유저를 위한 설계

불필요한 기능 없이, 매일 LLM을 사용하기 위한 빠르고 강력한 인터페이스.

나만의 API 키 사용

API 키는 iOS Keychain에 안전하게 저장됩니다. 중간 서버 없음, 백엔드 없음, 계정 불필요.

실시간 스트리밍

SSE를 통해 토큰 단위로 스트리밍 수신. 언제든 생성 중단 가능. 프로바이더별 전용 파서로 최고의 정확도.

세션 관리

모든 대화가 로컬에 저장됩니다. 세션 중 모델 전환 가능. 이름 변경, 삭제, 이어서 대화하기.

풍부한 Markdown

구문 강조가 적용된 코드 블록, 테이블, 리스트, 원탭 복사. 기술적 대화에 최적화.

비용 추적 계획 중

토큰 사용량과 모델 가격을 기반으로 턴별, 세션별 비용을 산출. 지출을 정확히 파악.

사고 수준 제어

세션별로 추론 깊이를 조절. 프로바이더를 아우르는 통합 추상화 — 최소 사고에서 확장 사고까지.

멀티 프로바이더, 하나의 인터페이스

사용하는 모델에 연결. 원활하게 전환.

OpenAI

Chat Completions & Responses API. 최신 OpenAI 모델에 연결.

Anthropic

Messages API. Claude Opus, Sonnet, Haiku 모델.

로컬 추론

Ollama, llama.cpp, vLLM 및 모든 OpenAI 호환 로컬 엔드포인트.
