개요

Arlo Lite(이하 "본 앱")는 대규모 언어 모델(LLM)과 상호작용하기 위한 무료 오픈소스 iOS 애플리케이션입니다. 우리는 사용자의 개인정보 보호에 최선을 다하고 있습니다. 본 방침은 본 앱이 처리하는 데이터, 저장 방법, 그리고 수집하지 않는 데이터에 대해 설명합니다.

수집하지 않는 데이터

  • 외부 서버에 개인정보를 수집, 전송 또는 저장하지 않습니다.
  • 어떠한 형태의 분석 도구, 텔레메트리 또는 추적 기술도 사용하지 않습니다.
  • 백엔드 서버가 없습니다 — 서버 측 구성 요소가 존재하지 않습니다.
  • 사용자의 대화를 열람하거나 제3자와 공유하지 않습니다.
  • 광고를 표시하지 않습니다.

기기에 저장되는 데이터

Arlo Lite에서 생성되는 모든 데이터는 사용자의 기기에만 저장됩니다:

  • 대화 기록 — 기기 내 SQLite 데이터베이스에 로컬 저장됩니다.
  • API 키 — iOS Keychain(Apple의 암호화된 자격 증명 저장소)에만 저장됩니다. 일반 텍스트 파일이나 암호화되지 않은 저장소에는 절대 기록되지 않습니다.
  • 프로바이더 설정 — 기기에 로컬 저장됩니다(모델 선택, 기본 URL, 표시 이름).
  • 사용자 환경설정 — 테마, 언어, 시스템 프롬프트 및 기타 설정이 로컬에 저장됩니다.

네트워크 통신

Arlo Lite는 다음의 경우에만 네트워크 통신을 수행합니다:

  • LLM API 호출 — 메시지를 전송하면 설정된 LLM 프로바이더(OpenAI, Anthropic 또는 사용자 정의 엔드포인트)로 직접 HTTPS 요청을 보냅니다. 대화 내용은 사용자가 선택한 프로바이더에 직접 전달되며, 본 앱은 이 통신에 관여하지 않습니다.
  • 모델 메타데이터 — 모델 정보를 최신 상태로 유지하기 위해 공개 CDN에서 모델 가격 및 컨텍스트 윈도우 크기 등의 정보를 가져올 수 있습니다.

선택한 LLM 프로바이더가 사용자의 데이터를 어떻게 처리하는지에 대해서는 해당 프로바이더의 개인정보 처리방침을 확인하시기 바랍니다.

iCloud 백업

앱에서 iCloud 백업을 활성화한 경우:

  • 대화 텍스트와 프로바이더 설정(API 키 제외)이 iCloud 계정에 백업될 수 있습니다.
  • API 키는 iCloud 백업에 포함되지 않습니다.
  • 파일이나 이미지 첨부 파일은 동기화되지 않습니다.
  • iCloud 데이터는 Apple의 개인정보 처리방침 및 사용자의 iCloud 계정 설정에 따릅니다.

제3자 서비스

본 앱은 분석, 크래시 리포트 또는 광고를 위한 제3자 SDK를 일체 포함하지 않습니다. 앱이 통신하는 외부 서비스는 다음뿐입니다:

  • 사용자가 명시적으로 설정한 LLM 프로바이더(OpenAI, Anthropic 또는 사용자 정의 엔드포인트).
  • Apple의 iCloud 서비스(백업을 활성화한 경우에만).

아동 개인정보 보호

Arlo Lite는 만 13세 미만 아동을 대상으로 하지 않습니다. 아동의 데이터를 의도적으로 수집하지 않습니다.

방침 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 변경 사항은 이 페이지에 반영되며 "최종 업데이트" 날짜가 갱신됩니다. 본 앱은 데이터를 수집하지 않으므로 변경은 최소한에 그칠 것으로 예상됩니다.
