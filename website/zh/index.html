Arlo Lite 是一款輕量級 iOS 客戶端，用於與大型語言模型對話。使用自己的 API Key 直連服務商——無後端伺服器、無訂閱、無遙測追蹤。

在 GitHub 查看 了解更多

為進階使用者打造

沒有多餘功能，只有快速、強大的 LLM 日常使用介面。

自帶 API Key

你的 API 金鑰安全存放在 iOS Keychain 中。沒有中間商、沒有後端伺服器、不需建立帳號。

即時串流回應

透過 SSE 逐字串流接收回覆。隨時可停止生成。各服務商獨立解析器確保最高還原度。

對話管理

所有對話記錄本地保存。可在對話中切換模型、重新命名、刪除或繼續先前的對話。

豐富 Markdown 渲染

程式碼區塊附語法高亮、表格、列表，一鍵複製。專為技術對話設計。

費用追蹤 計劃中

根據 Token 使用量與模型定價計算每則訊息和每段對話的花費，讓你清楚掌握開支。

思考力度控制

逐對話調整推理力度。跨服務商統一抽象——從最小到深度思考模式。

多服務商，統一介面

連接你慣用的模型，無縫切換。

OpenAI

Chat Completions 與 Responses API。連接最新的 OpenAI 模型。

Anthropic

Messages API。支援 Claude Opus、Sonnet 和 Haiku 模型。

本地推論

Ollama、llama.cpp、vLLM，以及任何 OpenAI 相容的本地端點。

隱私優先設計

無後端伺服器——直接呼叫服務商 API
無遙測或分析追蹤
API 金鑰僅存於 iOS Keychain
無需帳號或註冊
離線可閱覽歷史對話
可選 iCloud 備份（僅文字，不含金鑰）

永遠開源

Arlo Lite 可自由使用、修改和散佈。完整程式碼公開於 GitHub。使用 React Native 和 Expo 打造原生 iOS 體驗。
