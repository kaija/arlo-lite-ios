Arlo Lite は LLM と対話するための軽量 iOS クライアントです。自分の API キーを使ってプロバイダーに直接接続——バックエンドなし、サブスクリプションなし、テレメトリーなし。

GitHub で見る 詳しく見る

パワーユーザーのために設計

余計な機能なし。LLM を毎日使うための高速で高機能なインターフェース。

自分の API キーを使用

API キーは iOS Keychain に安全に保存。中間業者なし、バックエンドサーバーなし、アカウント不要。

リアルタイムストリーミング

SSE によるトークン単位のストリーミング。いつでも生成を停止可能。プロバイダー固有のパーサーで最高の忠実度。

セッション管理

全チャットをローカルに保存。セッション中にモデルを切り替え。名前変更、削除、続きから再開。

リッチ Markdown

シンタックスハイライト付きコードブロック、テーブル、リスト、ワンタップコピー。技術的な会話に最適。

コスト追跡 計画中

トークン使用量とモデル価格からターンごと・セッションごとのコストを算出。支出を正確に把握。

思考レベル制御

セッションごとに推論の深さを調整。プロバイダーを横断した統一的な抽象化——最小から拡張思考まで。

マルチプロバイダー、ひとつのインターフェース

使いたいモデルに接続。シームレスに切り替え。

プライバシー・バイ・デザイン

バックエンドサーバーなし——プロバイダーに直接通信
テレメトリーやアナリティクスなし
API キーは iOS Keychain のみに保存
アカウント・サインアップ不要
オフラインで過去の会話を閲覧可能
オプションの iCloud バックアップ（テキストのみ、キーは除外）
